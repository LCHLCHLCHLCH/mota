# 魔塔
## 简介
这是一个由C++开发的魔塔。
## 编译指南
### windows
1. 确保您已经安装了MinGW和cmake,并添加环境变量。
2. 将项目克隆到本地:
```bash
git clone https://github.com/LCHLCHLCHLCH/mota
```
3. 创建并进入build文件夹:
```bash
mkdir build
cd build
```
4. 执行cmake命令:
```bash
cmake -G "MinGW Makefiles" ..
```
5. 执行编译命令:
```bash
cmake --build .
```
